# Smart-Alert-and-Assistance-System-for-Visually-Impaired-Pedestrians

Système embarqué Arduino pour améliorer la sécurité des personnes malvoyantes aux passages piétons.
Détection ultrasonique des obstacles avec alertes visuelles, auditives et tactiles.
Projet démontrant l’usage des systèmes embarqués pour accessibilité et sécurité.
